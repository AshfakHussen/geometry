Enlighten Yourself with Some insights about DOM!

1.What is Dom

Answer: Dom is something that takes the html code and then represent it as a tree structure.

2.What is the purpose of Dom?

The purpose of DOM is to help programs and script to access, modify and update the content.

3.How to select element from Dom?

There are a number of ways to select element from DOM.Such as:
  • querySelector
  • querySelectorAll
  • getElementById
  • getElementByClassNames
  • getElementByTagNames

4.What is event Bubble?

Event Bubble is the concept where if a button is clicked, then it the event(click) will propogate all the way up to root parent step by step.
